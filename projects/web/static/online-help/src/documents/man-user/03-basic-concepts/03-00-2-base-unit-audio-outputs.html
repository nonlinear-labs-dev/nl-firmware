Audio Outputs

The audio output provides two parallel line-level stereo pairs of audio sockets (6.3mm TRS and XLR).

The signals are transformer-balanced and ground-free, therefore in most cases a DI-box is not necessary.

Unbalanced and balanced plugs can be connected. When connecting to unbalanced inputs please use unbalanced cables and plugs.

Note that using both TRS and XLR outputs at the same time may cause hum.
