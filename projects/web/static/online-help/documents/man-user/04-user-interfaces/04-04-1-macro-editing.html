Fast Mapping

Press the Selection Button for a potential target parameter while holding down the Selection Button for a Control to switch the assignment on or off. This only works for target parameters at the top of the stack.

This method only handles the actual mapping of target parameters to a Control. Modulation amounts can only be adjusted by focussing on each parameter. However, the modulation amounts are kept, and the last value they were adjusted to is recalled.
