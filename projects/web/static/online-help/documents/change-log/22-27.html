22-27 Service Update - new Hardware Support

Newer C15 devices (starting with serial no. 223) contain new hardware components that require software support and are not compatible with older software versions (before 22-27).

This update provides software support for new hardware components and ensures that C15 devices with new hardware components cannot be updated to earlier versions. For C15 devices with new hardware components, this update is essential. For older C15 devices, this update is optional.

We strongly recommend to not update newer C15 devices with older software versions, as the device will not be usable afterwards.
