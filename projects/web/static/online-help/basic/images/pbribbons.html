Bender & Ribbons

back to Overview

The two 800 mm Ribbons (touch strips) are equipped with 33 LED dots. They can be used as Hardware Source in absolute or relative mode, each combinable with individual return behavior (zero or center).
The magnetical loaded Bender can be used for bending pitches or other parameters. Read how to assign a Macro Control to Hardware Sources here.
